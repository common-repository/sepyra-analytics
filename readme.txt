== Description ==

This plugin adds the required javascript for sepyra analytics.

For more information visit:

(http://sepyra.com/about)

== Installation ==

This section describes how to install the plugin and get it working.

e.g.
1. Upload `sepyra-analytics` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the unique ID of your site from Sepyra Analytics to the settings (Admin > Settings > Sepyra)
